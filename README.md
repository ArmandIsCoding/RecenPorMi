# 🙏 Recen Por Mí (Open Source)

> *"Recen por mí, no se olviden"* — **Papa Francisco**

**Recen Por Mí** es una plataforma web de código abierto diseñada para crear una red de apoyo espiritual y empatía digital. El proyecto nace como un espacio donde cualquier persona puede compartir sus intenciones, pedidos o agradecimientos, y recibir el acompañamiento de una comunidad que se compromete a rezar por ellos.

En lugar de "likes" superficiales, el sitio utiliza el emoji de oración (**🙏**) como motor de interacción: un gesto simple pero cargado de significado para avisar que apoyan y van a rezar por esa causa.

---

## ✨ Características Principales
* **Peticiones de Oración:** Publicá tus intenciones con total privacidad o bajo un pseudónimo.
* **Red de Oración Activa:** Cada interacción suma un apoyo real a la intención, enviando una señal de aliento al autor.
* **Privacidad y Respeto:** Diseñado para ser un espacio seguro y anónimo si el usuario lo prefiere.
* **Feed en Tiempo Real:** Visualización instantánea de nuevos pedidos y apoyos mediante **SignalR**.
* **Arquitectura Liviana:** Optimizado para ser hosteado en mini-servidores personales (IIS / Windows).

## 🛠️ Stack Tecnológico
Este proyecto está construido sobre el ecosistema de **.NET**:

* **Frontend/Backend:** [Blazor](https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor) (C#).
* **Acceso a Datos:** Entity Framework Core.
* **Base de Datos:** SQL Server.
* **Comunicación en vivo:** SignalR.
* **Hosting:** IIS sobre Windows Server.

## 🚀 Instalación y Despliegue (Próximamente)
1. Clonar el repositorio.
2. Configurar la cadena de conexión en `appsettings.json`.
3. Ejecutar las migraciones de base de datos.
4. Publicar en IIS o correr localmente con `dotnet run`.
